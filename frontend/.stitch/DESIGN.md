# Design System: Titan Neon Sport

Inspired by modern high-performance fitness apps, this design system uses a sharp "Neon Lime" accent against a deep neutral dark background.

## Palette

- **Background (Main)**: `#050505` (Pure Black)
- **Surface (Cards/Grid)**: `#121212` (Dark Grey/Black)
- **Primary Accent**: `#B4FF00` (Neon Lime)
- **Secondary Accent**: `#222222` (Deep Grey for borders)
- **Text (Primary)**: `#FFFFFF` (White)
- **Text (Secondary)**: `#777777` (Muted Grey)

## Typography

- **Headings**: Archivo/Inter, ExtraBold Uppercase
- **Body**: Inter, Medium

## Elements

- **Bento Bricks**:
  - Cards with large border-radius (24px - 32px).
  - Subtle 1px borders (`#222222`).
  - Some blocks use the solid Neon Lime background with black text.
- **Buttons**:
  - Pill-shaped (fully rounded).
  - Primary: Neon Lime background with Black text.
  - Secondary: Dark grey with white text and an arrow icon.
- **Grids**:
  - Use of CSS Grid for a "Bento Box" dashboard and landing page.

## Atmosphere

- High-tech, performance-driven, and energetic.
- Minimalist iconography.
- High-contrast elements.
